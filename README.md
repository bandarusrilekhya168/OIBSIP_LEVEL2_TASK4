# Login Authentication System

## Oasis Infobyte Web Development and Designing Internship

### Level 2 - Task 4

A simple Login Authentication System developed using HTML, CSS and JavaScript. This project allows users to register, login, access a protected dashboard and logout securely using browser Local Storage.

## Features

- User Registration
- User Login
- Credential Validation
- Protected Dashboard Access
- Logout Functionality
- Local Storage Integration
- Responsive User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

├── script.js

└── README.md

## How to Use

1. Open the application.
2. Register a new account.
3. Login using the registered credentials.
4. Access the dashboard.
5. Logout securely.
